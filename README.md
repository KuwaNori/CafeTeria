# CafeTeria
--Ishibashi Seminor Project of July--
### Mission
フードロス削減と、学生の健康な食生活を目指して立ち上がったプロジェクト。<br>
ひとり暮らしで生活が厳しい学生と、廃棄される食材を有効に消費したい学食をターゲットにWebアプリケーションを作成した。<br>
